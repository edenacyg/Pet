# My Pet

An application that sets the name of your pet.

## 📃 How to play:
1. Provide a name for your pet on the textarea.
2. The provided name will display on the bottom text.

## 🛠 Tools used:
• HTML

• CSS

• React JS

• Node JS

• Express

• JSX(JavaScript Extension)

## 💡 What I learned:
• React Hooks.

• Understand what is localstorage and how it work.
